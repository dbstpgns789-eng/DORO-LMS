프로젝트 목표

딥러닝 기반 영상 분석 기술을 활용하여 상업 공간(예: 영화관)의 방문객
연령대 및 구매 관련 행동 데이터를 자동으로 추정하고, 이를 실시간으로
관리자/운영자가 확인할 수 있는 상업용 데이터 분석 시스템을 구축하는 것이
목표이다. 이 데이터는 마케팅, 고객 분석, 운영 최적화 등에 활용될 수
있다.

1단계: 기초 모델 개발

사용 데이터: - UTKFace 얼굴 데이터셋

모델 구조: - ResNet50 기반 분류 모델 - UTKFace로 나이 분류 성능을 목표로
기본 학습 및 1차 미세 조정(Fine-tuning) 진행 - 현재까지는 이 단계까지
완료됨

2단계: 실사용 데이터 확보 및 모델 고도화(예정)

실제 환경 데이터 수집: - 영화관 등 상업 공간에서 군중 이미지/영상 데이터
수집 - 각 프레임에서 얼굴 검출 후 개별 얼굴 이미지 추출 - 실제
환경에서는 모자, 선글라스, 조명 등 복잡한 요소 존재

데이터 라벨링: - 실제 군중 이미지에는 라벨이 없음 → 직접 라벨링 필요

2차 모델 미세 조정: - 실제 환경에서 수집한 얼굴 데이터로 재학습 및
도메인 적응

문제점 및 주의사항

1.  데이터 간 도메인 차이
2.  라벨링 비용 증가
3.  얼굴 가려짐·차폐 문제
4.  실시간 시스템 구현 시 속도 최적화 필요
